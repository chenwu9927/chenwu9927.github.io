开始之前

本文档讲述了使用Hexo搭建静态个人博客并将其部署在Github Pages上的详细过程。此外，还讲述了使用自定义域名、更换主题、使用swiftype搜索引擎以及部署在vercel上的过程。

安装Git

Hexo构架博客、推送更新需要通过git命令行来操作。Windows操作系统可以直接从官网下载并安装git。

之后若要在某路径下使用git bash，可在该目录单击右键 - Git Bash Here 或 使用cd <path>命令进入该路径。

安装Node.js

要使用Hexo框架，安装Node.js环境。进入官网下载适合你的操作系统的安装包并安装。

创建Github仓库

登录Github，单击头像 - Your repositories 然后在转跳后的页面中单击new。

在Repository name 文本框中填写 <user_name>.github.io 。 <user_name>.github.io便是此后github pages的网址。其中<user_name>是你的Github用户名。

务必勾选Public 。因为只有公共仓库能使用github pages。

然后单击 Create repository 完成创建。
